# 全球短视频趋势中心 V2

固定地址：

`https://qiucheng777.github.io/global-short-video-report/`

## 已实现

- 每天土耳其时间 16:00 自动运行
- 自动采集公开网页和 RSS 趋势信号
- 国外 Top10 与中国 Top10
- 自动趋势摘要与拍摄选题
- 每日 JSON 历史归档
- 固定链接不变
- 搜索、图表、手机适配
- 数据不足时明确标注，不伪造榜单

## 覆盖安装

将本压缩包解压后，把里面所有文件上传到原仓库根目录，选择覆盖同名文件并 Commit。

## 首次立即运行

仓库顶部进入 `Actions` → `Update global short-video report` → `Run workflow`。

以后每天自动运行，无需手动操作。
